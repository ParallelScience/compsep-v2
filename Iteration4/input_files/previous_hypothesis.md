The current SR-DAE architecture, while successful at reconstructing tSZ power, suffers from "intensity-dependent bias" in massive clusters because the model treats the tSZ signal as a monolithic regression target. I hypothesize that decomposing the tSZ signal into a "Morphological Base" (large-scale gas distribution) and a "Feedback Residual" (small-scale AGN-driven fluctuations) using a multi-task learning framework will mitigate the peak-intensity attenuation. Specifically, by training the U-Net to simultaneously predict the tSZ map and a secondary "Feedback Intensity" map (derived from the high-pass filtered ground truth), the model will be forced to decouple the global cluster profile from the high-frequency feedback signatures. This dual-head approach, combined with a "Gradient-Weighted Loss" that dynamically increases the penalty for errors in high-gradient regions (cluster cores), will allow the network to preserve the peak intensity of massive clusters without sacrificing the small-scale power recovery achieved in previous iterations.
